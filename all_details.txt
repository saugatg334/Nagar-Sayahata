## 🔐 Authentication System

### **Current Features**
- **Login/Logout**: Standard Django authentication
- **Brute Force Protection**: IP-based rate limiting with progressive delays
- **Session Management**: Secure session handling
- **Security Features**:
  - CSRF protection enabled
  - Session timeout (1 hour)
  - IP tracking for failed attempts
  - Progressive delay on failures (0.5s, 1s, 2s, 3s, 4s)
  - Lockout after 5 failed attempts (20 seconds)

### **Authentication Flow**
1. User accesses `/login/`
2. System checks for existing lockout
3. Validates credentials
4. On success: Reset attempts, create session, redirect to home
5. On failure: Record attempt, apply delay, show generic error

### **Security Implementation**
```python
# Key security features in views.py
MAX_ATTEMPTS = 5
LOCKOUT_TIME = 20  # seconds
CACHE_TIMEOUT = 300  # 5 minutes

# IP-based tracking
ip_address = request.META.get('REMOTE_ADDR')
cache_key = f'login_attempts_{ip_address}'
```

## 📄 Template System

### **Template Hierarchy**
```
templates/
├── admin_base.html          # Base template with blocks
├── admin_pannel.html        # Main layout (sidebar + navbar)
├── authentication/
│   └── login.html           # Login page (standalone)
└── home/
    └── home.html            # Dashboard content
```

### **Template Inheritance Flow**
1. **admin_base.html**: Defines basic HTML structure and includes admin_pannel.html
2. **admin_pannel.html**: Defines sidebar, navbar, and content block
3. **home.html**: Extends admin_base.html and fills content block

### **Static File Loading**
```html
<!-- CSS Loading -->
<link rel="stylesheet" href="{% static 'css/admin_pannel/style.css' %}">

<!-- JavaScript Loading -->
<script src="{% static 'js/admin_pannel/script.js' %}"></script>
```
